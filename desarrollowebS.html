¿Como se gestiona un proyecto web?


Proceso de desarrollo de un proyecto
Analisis


El principal objetivo de esta fase es realizar un análisis detallado de los requerimientos y de las necesidades para asegurar la viabilidad de desarrollar dicho proyecto.

Diseño


Luego se establece la estructura lógico-física del proyecto para proveer bases estables y permitir la construcción de los módulos pertinentes al requerimiento.

Desarrollo


Se continua con clarificar los requerimientos y se completa el desarrollo del proyecto, el cual de cumplimiento a los requerimientos del mismo.

Transición


La última fase comprende las pruebas del proyecto para asegurar que el mismo esté disponible y en completo funcionamiento para proceder con la implementación.
